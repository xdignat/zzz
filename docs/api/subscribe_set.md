## subscribe_set
Устанавливает данные подписки.

### Параметры:
- ```token``` (string, required) - токен сессии
- ```rubric``` (string) - имя рубрики 
- ```subscribe``` (string) - id подписки 
- ```user_name``` (string) - имя пользователя
```json
{
  "token": "2423fdsf",
  "subscribe": "436de39f",
  "values": {
    "user_name": "Иван Иваныч"
  }
}
```
