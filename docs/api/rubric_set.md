## rubric_set
Устанавливает даные рубрики.

### Параметры:
- ```token``` (string, required) - токен сессии
- ```rubric``` (string) - имя рубрики
- ```description``` (string) - описание
```json
{
  "token": "2423fdsf",
  "rubric": "news", 
  "values": {
      "rubric": "news", 
      "description": "новости",
      "enable": true
  }
}
```
